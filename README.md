# 基于强化学习的井字棋 AI

一个使用时序差分(TD)强化学习算法训练的井字棋 AI 系统。通过自我对弈，AI 学习最优的游戏策略。

## 项目概述

这个项目实现了一个井字棋 AI，使用强化学习中的时序差分法(Temporal Difference Learning)来训练。AI 通过与随机对手进行数百万局对弈，逐步学习最优的下棋策略。

### 核心特性

- **强化学习算法**：使用时序差分(TD)学习方法
- **ε-贪心策略**：平衡探索(Exploration)和利用(Exploitation)
- **价值表存储**：将所有可能的棋盘状态映射到价值评估
- **无头训练模式**：支持高速训练（无图形界面）
- **动态 Epsilon 衰减**：平滑的探索到利用的过渡

## 工作原理

### 1. 棋盘表示

- 使用 9 元素数组表示 3×3 棋盘
- 每个格子的值：0(空) / 1(AI1) / 2(AI2)
- 棋盘状态转换为 3 进制数作为价值表的索引
- 总共 3^9 = 19,683 种可能的棋盘状态

### 2. 强化学习算法

#### 时序差分(TD)学习

使用 TD(0) 算法更新价值估计：

```
V(s) = V(s) + α * [V(s') - V(s)]
```

其中：
- `V(s)` 是当前状态的价值估计
- `V(s')` 是下一个状态的价值估计
- `α` 是学习率（控制更新步长）

#### ε-贪心策略

在每一步决策时：
- 以 ε 的概率随机选择一个位置（探索）
- 以 1-ε 的概率选择价值最高的位置（利用）

这样可以平衡：
- **探索**：发现新的策略
- **利用**：使用已知的最优策略

### 3. 训练过程

1. **初始化**：两个 AI 智能体，价值表初始值为 0.5
2. **对弈**：AI1 和 AI2 进行一局游戏
3. **学习**：根据游戏结果更新 AI1 的价值表
4. **重复**：进行数百万局对弈，逐步改进策略

### 4. 奖励设计

游戏结束时的奖励：
- **AI1 赢**：AI1 获得 +1，AI2 获得 -1
- **平局**：双方都获得 0
- **AI1 输**：AI1 获得 -1，AI2 获得 +1

## 关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 总训练局数 | 5000 万 | 训练的游戏局数 |
| 初始 Epsilon | 0.5 | 初始探索率（50%随机） |
| 最小 Epsilon | 0.05 | 最终探索率（5%随机） |
| 学习率(α) | 0.1 | TD 更新的步长 |
| Epsilon 衰减 | 0.9999 | 每局衰减系数 |

## 使用方法

### 编译

```bash
# 使用 g++ 编译（需要 EasyX 图形库）
g++ -o ooxx main.cpp ai.cpp board.cpp -leasyx
```

### 运行

#### 无头训练模式（推荐）

编辑 `main.cpp`，设置：
```cpp
const bool ENABLE_GRAPHICS = false;  // 禁用图形
```

然后运行：
```bash
./ooxx
```

这样会以最快速度进行训练，不显示图形界面。

#### 带图形训练模式

编辑 `main.cpp`，设置：
```cpp
const bool ENABLE_GRAPHICS = true;  // 启用图形
```

然后运行：
```bash
./ooxx
```

这样会实时显示棋盘和棋子，但训练速度会慢很多。

### 输出

训练过程中每 50 万局会输出一次统计信息：

```
第 500000 局 | AI1胜: 250000 (50.00%) | AI2胜: 150000 (30.00%) | 平局: 100000 (20.00%) | Epsilon: 0.4950
```

训练完成后输出最终统计：

```
========== 训练完成 ==========
总局数: 50000000
AI1胜利: 25000000 (50.00%)
AI2胜利: 15000000 (30.00%)
平局: 10000000 (20.00%)
=============================
```

## 代码优化

### 性能优化

1. **静态随机数生成器**
   - 避免每次调用都重新初始化 RNG
   - 显著提升训练速度

2. **无头训练模式**
   - 禁用图形绘制，专注于算法训练
   - 训练速度提升 10 倍以上

3. **内存高效**
   - 使用二进制文件存储价值表
   - 19,683 个 double 值约占 157 KB

### 算法改进

1. **平滑的 Epsilon 衰减**
   - 使用指数衰减而不是阶跃衰减
   - 更平稳的探索到利用的过渡

2. **改进的 TD 更新**
   - 正确处理游戏结束时的价值更新
   - 确保最后一步的学习信号正确传播

3. **详细的注释**
   - 所有函数都有中文注释
   - 算法原理清晰易懂

## 文件说明

### main.cpp
- 主程序入口
- 训练循环的实现
- 统计信息的输出
- 支持无头训练模式

### ai.h / ai.cpp
- `ai` 类：AI 智能体的实现
- `move()`：ε-贪心决策函数
- `convertBoard()`：棋盘状态转换为索引
- `saveValueTable()` / `loadValueTable()`：价值表的持久化

### board.h / board.cpp
- 棋盘管理和游戏逻辑
- `initBoard()`：初始化棋盘
- `drawBoard()` / `drawPiece()`：图形绘制
- `checkWin()`：检查游戏是否结束
- `getAvailble()`：获取可用位置

### value.bin
- 二进制格式的价值表
- 存储 19,683 个 double 值
- 可以被加载用于继续训练或测试
